## 🚀 About ToolMight

**ToolMight** is a lightweight collection of developer tools designed to make everyday development tasks faster and easier.

No clutter. No login. Just useful tools.

Whether you're formatting JSON, testing regex, or encoding data, ToolMight provides a simple interface to get the job done quickly.

---

## 🧰 Available Tools

Some of the utilities currently available on ToolMight:

- JSON Formatter  
- Regex Tester  
- Base64 Encoder / Decoder  
- URL Encoder / Decoder  
- Hash Generator  

More tools are being added regularly.
